Description
tListe est une applet de liste hiérarchique (TreeView). Sa taille réduite vous permet de l'utiliser facilement (entre 12 et 20 Ko suivant la version). Elle est totalement paramétrable et fonctionne sur tous les navigateurs supportant le JDK 1.02.
Cette dernière version n'est disponible qu'en version JDK 1.1 (La version JDK 1.02 est la 6.01).

Paramètres
Param name Type Description
file string fichier à utiliser ou préfixe des paramètres
autoclose char "Y" si fermeture des noeuds lors d'un click sur un nouveau noeud.
acpress integer mouseoever presentation
  • 0 : pas de changement
  • 1 : défaut (bordude 3D)
  • 2 : changement de couleur
  • 3 : soulignement
  • 4 : changement de couleur et soulignement
accolor integer couleur lors de l'action "mouseoever" (si acpress = 2 or 4) (en hexa si commence par "#")
bImg string Nom du fichier des icônes "./img/icon3.gif
bgImg string Nom de l'image de fond
bgcolor integer Couleur du fond
target string "target" windows de défaut
csep string caractère séparateur du fichier de données (défaut ";")
audio string répertoire pour les fichiers son (.au)
local string "yes" si vous placez les données en paramètres à partir de la version 6.15
encoding string Encoding definition pour le fichier texte (défaut Cp1252) à partir de la version 6.15
coef integer Taille d'une ligne en %
v0..9 string Definition des variables
avec cela vous pouvez réduire la taille du fichier de données
si v0 = "http://www." et vous definissez das le fichier : $0yahoo.com l'applet tliste comprend http://www.yahoo.com
font0 Bullet style Font Name, font style, size, font color, bgcolor
font1..9 Font style Font Name, font style, size, font color, visited color
!! Attention !! vous devez convertir vos codes couleur en décimal avant de les utiliser.
Exemple :
white => #FFFFFF => 16777215
red => #FF0000 => 16711680
See http://www.berthou.com/color_code.html

Data File Format
Colum Name Type Description
1 Level Integer Item level 0..20
Remarques :
98 Button Bar image (with tListe V4.0)
99 Bitmap image
2 Item Name string Name of item (URL of bitmap if level 99)
3 Bitmap number Integer Item bitmap (0..9)
Rem : MOD(2) = 0 => treenode close (if node)
MOD(2) = 1 => treenode open (if node)
2 node disable.
sample if node "0, 10,12,20,..." => node close
"1, 11,13,21,..." => node open
View tListe Image library
3b Font style int Selected font and color style
4 Dest URL string Dest. URL ( begin "./" if local URL )
5 Target string Target Frame (overide default in PARAM option)
Rem : to load another datafile use "_reload"
if "_" use default target
if "_reload" load another file
if "_prev" load a previos file (not use url column)
if "_insert" insert a file
if "_script" call javascript function
6 Date New YYYYMMDD Last day to display "new" bitmap
- Help Bullet string Begin with "?" String help display in bullet, you can define this param in 5, 6 or 7 col.
- Help Status string Begin with "#" String help display status bar, you can define this param in 5, 6 or 7 col.
- Key-words string Begin with "key=" key-words definition for javascript use.

Use JavaScript
With tListe you can call external javascript function (with LiveConnect and MAYSCRIPT tag)
_script With this you can call javascript function when you click on tliste item
1 § Alert-2 § 7 § javascript:alert('You click on Alert-2') § _script
And you can call tListe function in JavaScript...
You can call :
rFile("yourfile.txt", 1) To load another data file.
closeAll( 0 ) to close all item folder.
closeAll( 1 ) to open all item folder.
change('Outils', i) to open or close 'Outils' item and see this in 1st position
if "i" equals "1" the url is open
setBgcolor( int rgbcolor ) to change background color.
search('toto', i) search key-word "toto" and open this item if "i" is equals "1" (if = "2" not open but set arrow) .
next('toto', i) search "next" key-word "toto" and open this item if "i" is equals "1" .
